# Low Level Design (LLD) Interview Preparation

This repository contains my Low Level Design practice implementations in Java, focused on mastering OOP principles, SOLID principles, design patterns, and real-world interview-style system design problems.

---

## 📌 Goals

- Strengthen Object-Oriented Design skills
- Apply SOLID principles in real systems
- Implement common Design Patterns
- Solve real-world LLD interview problems
- Write clean, extensible, and maintainable code

---

## 🧠 Concepts Covered

- Object-Oriented Programming (OOP)
- SOLID Principles
- Composition over Inheritance
- Encapsulation & Abstraction
- Interface-driven design
- Clean Code Practices

---

## 🎨 Design Patterns Implemented

- Strategy
- Observer
- Factory & Abstract Factory
- Singleton
- Decorator
- State
- Command
- Adapter
- Template Method

Each pattern is implemented with:
- Clear class structure
- Example use case
- Explanation of when to use it

---

## 🏗 LLD Problems Implemented

- Parking Lot System
- Vending Machine
- Splitwise
- Elevator System
- Tic Tac Toe
- BookMyShow
- Snake & Ladder

Each problem includes:
- Requirement assumptions
- Proper class separation
- Extensible design
- Application of design patterns where relevant
